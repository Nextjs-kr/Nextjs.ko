---

Next.js는 `x-powered-by` 헤더를 기본으로 추가합니다. 기본으로 추가되는 헤더를 제외하려면 `next.config.js` 파일을 열어 `poweredByHeader`를 비활성화 합니다.

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
