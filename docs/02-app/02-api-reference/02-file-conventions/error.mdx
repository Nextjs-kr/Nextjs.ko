**error** 파일은 라우트 세그먼트의 error UI를 정의합니다.

```tsx filename="app/dashboard/error.tsx" switcher
'use client' // Error 컴포넌트는 Client 컴포넌트여야 합니다.

import { useEffect } from 'react'

export default function Error({
  error,
  reset,
}: {
  error: Error
  reset: () => void
}) {
  useEffect(() => {
    // 오류 리포트 서비스에 오류를 기록합니다.
    console.error(error)
  }, [error])

  return (
    <div>
      <h2>Something went wrong!</h2>
      <button
        onClick={
          // 세그먼트를 리렌더링하여 복구를 시도합니다.
          () => reset()
        }
      >
        Try again
      </button>
    </div>
  )
}
```

```jsx filename="app/dashboard/error.js" switcher
'use client' // Error 컴포넌트는 Client 컴포넌트여야 합니다.

import { useEffect } from 'react'

    <div>

        onClick={
          // 세그먼트를 리렌더링하여 복구를 시도합니다.
          () => reset()
        }
      >
        Try again
      </button>
    </div>
  )
}
```

## Props

### `error`

[`Error`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Error) 객체의 인스턴스입니다. 이 오류는 서버 또는 클라이언트에서 발생할 수 있습니다.

### `reset`

error boundary를 재설정하고, response를 반환하지 않는 함수입니다.

> **참고**:
>
> - `error.js`는 **[Client 컴포넌트](/docs/getting-started/react-essentials)** 여야 합니다.
> - error boundary가 해당 레이아웃 컴포넌트 **안에** 중첩되어 있기 때문에 `error.js`는 **동일한** 세그먼트의 `layout.js` 컴포넌트에서 발생한 오류를 **처리하지 않습니다.**
>   - 특정 레아아웃의 오류를 처리하면 `error.js` 파일을 레이아웃 상위 세그먼트에 배치합니다.
>   - 루트 레이아웃 또는 템플릿 내에서 오류를 처리하려면 `error.js`를 변형한 `app/global-error.js`를 사용하세요.

## `global-error.js`

루트 `layout.js`의 오류를 명확하게 처리하려면 루트 `app` 디렉토리에 있는 `error.js`를 변형한 `app/global-error.js`를 사용합니다.

```tsx filename="app/global-error.tsx" switcher
'use client'

export default function GlobalError({
  error,
  reset,
}: {
  error: Error
  reset: () => void
}) {
  return (
    <html>
      <body>
        <h2>Something went wrong!</h2>
        <button onClick={() => reset()}>Try again</button>
      </body>
    </html>
  )
}
```

```jsx filename="app/global-error.js" switcher
'use client'

export default function GlobalError({ error, reset }) {
  return (
    <html>
      <body>
        <h2>Something went wrong!</h2>
        <button onClick={() => reset()}>Try again</button>
      </body>
    </html>
  )
}
```

> **참고**:
>
> - `global-error.js`가 활성화되면 루트 `layout.js`를 대체하므로 자체 `<html>` 및 `<body>` 태그를 정의해야 합니다.
> - error UI를 디자인할 때, [React Developer Tools](https://react.dev/learn/react-developer-tools)를 사용하는 것이 도움이 될 수 있습니다.
